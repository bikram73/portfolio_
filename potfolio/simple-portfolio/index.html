Facial Recognition and Attendance System

Brahmi to Kannada Image Text Translation Project

Developed an advanced image-to-text translation system to convert Brahmi script in images into modern Kannada text. The project utilized Optical Character Recognition (OCR) techniques with Python libraries like OpenCV and Tesseract for script detection and recognition. A custom mapping algorithm translated recognized Brahmi characters into Kannada. This project is a valuable tool for historians and linguists, enabling efficient transcription of ancient manuscripts while preserving cultural and linguistic heritage.
